### Description
`rc_devcoords` is a simple yet powerful tool for FiveM developers that allows players to display their current coordinates and heading in both the chat and on-screen. This script enhances developer efficiency by providing real-time information about player positioning within the game.

### Features
- **Toggleable Coordinate Display**: Easily enable or disable the display of coordinates.
- **Real-time Updates**: Coordinates are updated every 5 seconds when enabled.
- **Chat Integration**: Coordinates can be displayed in the chat for quick reference.
- **On-screen Display**: Coordinates are shown at the bottom right corner of the screen.

1. **Download the Script**: Clone or download this repository to your local machine.
2. **Add to Your Resources**: Place the `rc_devcoords` folder in your FiveM resources directory.
